= ユースケース図

大まかなユースケースを網羅することで登場するオブジェクトを洗い出す

ユースケースではユーザの操作のみ起点とし、サービス側の動きはシーケンス図の方で記載する？
ただし、他のユーザ操作に関わる動き（接続用ポートの待ち受けを開始するなど）はユースケース内にも記載する。

- ユースケースはコンセプトに反映
- シーケンスやアクティビティはリファレンスに反映
