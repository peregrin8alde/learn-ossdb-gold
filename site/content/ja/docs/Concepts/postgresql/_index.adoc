== PostgreSQL とは

PostgreSQL はオープンソースのオブジェクトリレーショナルデータベース管理システム（ ORDBMS ）です。

=== オブジェクトリレーショナルデータベース管理システム（ ORDBMS ）とは

リレーショナルデータベース管理システム（ RDBMS ）の一種であり、オブジェクト指向を取り込んでいる。
