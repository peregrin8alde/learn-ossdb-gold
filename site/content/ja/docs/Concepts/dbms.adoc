= データベースマネージメントシステム（ DBMS ）

データベースマネージメントシステム（ DBMS ）とは

データベース（データを蓄積する入れ物と蓄積されたデータ）を管理するシステム

クライアントからはデータの格納や参照の依頼
サーバからは依頼に対する各種メッセージや参照したデータ
